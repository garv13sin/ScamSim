**ScamSim** is an interactive cybersecurity awareness game powered by an **LLM (Large Language Model)** where players step into the role of a scammer and attempt to scam an AI through an online chat interface.

The goal of the game is to **understand real-world scamming techniques** and learn how scammers manipulate victims. After each interaction, the system provides **educational feedback explaining the mistakes made by the AI** and how similar scams can be identified and prevented in real life.

---

# 🚀 Features

### 🎮 Multiple Targets (Levels)

* Different **AI targets** representing different scenarios
* Each level increases in difficulty

### 🕵️ Target Intelligence

* Each target includes:
  * Background information
  * Behavioral hints
  * Potential vulnerabilities
